# UI Automation Framework 🚀

[![Automated UI Tests](https://github.com/Saurabh211096/ui-automation-framework/actions/workflows/ui-tests.yml/badge.svg)](https://github.com/Saurabh211096/ui-automation-framework/actions/workflows/ui-tests.yml)

An enterprise-ready page-object-model test automation framework using Java, Selenium Grid, TestNG, and ExtentReports.

## Tech Stack
* **Language:** Java 17
* **Testing Framework:** TestNG
* **Build Tool:** Maven
* **Reporting:** ExtentReports / Allure
* **CI/CD:** GitHub Actions

## Running Tests Locally
To run tests locally: `mvn clean test`
